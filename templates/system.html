RAG 检索流程

User Query
→
Generate Embedding
→
Vector Similarity Search
→
Retrieve Top-K Chunks
→
Build RAG Prompt
→
LLM Generation
→
Response
def rag_query(query: str) -> str: # 1. 向量化 query_embedding = embedding.encode(query) # 2. 向量检索 results = vector_store.similarity_search(query_embedding, k=3) # 3. 构建上下文 context = "\n".join([doc.page_content for doc in results]) # 4. 调用LLM response = llm.generate(context + query) return response
